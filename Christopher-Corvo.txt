    2. What is HTML used for?  
    
        HTML is a markup language used to layout text and other data.
        It provides the general structure of a page. Think of it as the wood frame of a house.
    
    3. What is an attribute and where do we put it? 
        
        An attribute is a word within an html tag that modify an HTML element. src for example is an attribute that modifies <img>

    4. What is the h1 tag used for? How many times should I use it on a page?

        An <h1> tag is a heading tag. You can use it for titles. <h1> is meant to be used only once for the main title.

        An id is an identifer you can add to an element to set it apart. You should only use an id once. As a rule of thumb use classes instead of id

        the class is an attribute used to identify an element specifically for css. 

    11. What elements can I add a class to? 

        Class can be added to any HTML element. 

    12. How many times can I use the same class on a page? 

        You can use a class multiple times on a page. 

    13. How do I get my link to open in a new tab?

        The way to get your link to open in a new page is by adding target="_blank" to <a href.

    14. What is the alt attribute in the image tag used for? 

        The alt atribute in an image tag provides a text alternative to an image. So if the image cant display it will show text.
        This is also important for screen reading devices and accesbility. 

    15. How do I reference an id?

    16. What is the difference between a section and a div

        <section> is more semanticaly rich way to divide information on a page. It means that the information included goes together or has some connection. 
        <div> breaks up info but it doesnt have any real meaning.
